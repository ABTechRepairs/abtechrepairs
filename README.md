# AB Tech Repairs Website

A simple static website for AB Tech Repairs.

## Files

- `index.html` — website content
- `styles.css` — design and mobile layout
- `script.js` — automatically updates the footer year
- `CNAME` — tells GitHub Pages to use `abtechrepairs.com`

## Publish with GitHub Pages

1. Create a new public GitHub repository, for example `ab-tech-repairs`.
2. Upload all four files from this folder to the root of the repository.
3. Open the repository's **Settings**.
4. Select **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/ (root)`, then save.
7. In the **Custom domain** box, enter `abtechrepairs.com` and save.
8. Configure the domain's DNS records with your domain provider.
9. Once available, enable **Enforce HTTPS**.
